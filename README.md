# Week 1 — My First Data Notebook

Exploring a movies dataset using pandas and NumPy.

## Files
- `movies.csv` — dataset of 50 top-rated films
- `week1_data_notebook.ipynb` — notebook that loads, cleans, and analyses the data

1. What is the average IMDb rating?
2. Which genre appears most often?
3. Which movie made the most money?
4. Who are the top 3 directors?
5. Do higher-rated movies run longer?
